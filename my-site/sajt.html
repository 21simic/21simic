HTML

- HTML je standardni jezik za kreiranje web stranica.
- HTML je skracenica za Hyper Text Markup Language.
- HTML opisuje strukturu web stranice pomocu oznaka-tagova.
- HTML elementi su gradjevinski materijal za izradu web stranice.
- HTML elementi su predstavljeni oznakama-tagovima
- HTML oznake oznacavaju delove sadrzaja kao sto su "naslov","paragraf","tabela" itd..
- Internet pretrazivaci ne prikazuju HTML oznake-tagove ali ih koriste za prikazivanje sadrzaja na webu.

Na sledecem primeru prikazan je jednostavan HTML dokument

<!DOCTYPE html> <html> <head> <title> Moj novi web sajt </title> </head> <body> <h1>Ovo je naslov web sajta</h1> <p>Ovo je neki tekst</p> </body> </html>

<!DOCTYPE html> - Ova deklaracija definise ovaj dokument kao HTML5.

<html> - Ovo je osnovni element html stranice.

<head> - Ovaj element sadrzi meta podatke o dokumentu.

<title> - Ovaj element odredjuje naslov stranice.

<body> - Ovaj element prikazuje vidljiv sadrzaj stranice.

<h1> - Ovaj element definise veliki naslov.

<p> - Ovaj element definise paragraf.

HTML Osnovni elementi

- Svaki HTML dokument treba da pocne deklaracijom !Doctype
- Svaki dokument pocinje tagom html a zavrsava se tagom /html
- Deo html sadrzaja koji je uvek vidljiv u pretrazivacu nalazi se izmedju tagova body i /body
- HTML naslovi se definisu od naslova najvece vaznosti (h1) pa do naslova najmanje vaznosti (h6).
- HTML paragrafi se definisu tagovima p
- HTML linkovi se definisu tagom a koji u sebi ima atribut href.
- HTML slike su definisane tagom img. Izvorni kod src, alternativni tekst alt i visina i sirina slike su dati kao atributi.

HTML Elementi

- HTML elementi se uglavnom sastoje od pocetnog i zavrsnog taga sa sadrzajem koji je umetnut izmedju njih.
Element je sve sto se nalazi izmedju startnog i zavrsnog taga.

<p>Ovo je paragraf i predstavlja html element</p>

- HTML elementi koji nemaju sadrzaj zovu se prazni elementi, prazni element nema zavrsni tag kao sto je npr br tag

HTML Atributi

1) Svi HTML elementi mogu imati atribute.
2) Atributi prizaju dodatne informacije o elementu.
3) Atributi su uvek naznaceni u pocetnoj oznaci.
4) Atributi obicno dolaze u parovima ime-vrednost, kao npr. ime="vrednost".

Atribut href

HTML linkovi su definisani tagom a. Adresa linka je navedena u atributu href

<a href="http://www.ets.edu.rs">Sajt skole</a>

Atribut src

Slike u html-u su definisane tagom img. Ime fajla koji bi predstavljao sliku navedeno je u atributu src

<img src="ime_slike.jpg">

Atributi width i height.

Dimenzija slike u html-u podesava se pomocu ovih atributa.

<img src="ime_slike.jpg" width="500" height="600">

Atribut alt

U atributu alt se nalazi alternativni tekst koji bi bio ispisan u koliko je slika obrisana sa servera ili iz nekog drugog razlogane moze da se ucita u pretrazivacu.

<img src="ime_slike.jpg" alt="alternativni tekst">

Atribut style

Atribut style se koristi za odredjivanje stila elemenata kao sto je boja, font, velicina itd.

<p style="color: blue;">Ovo je paragraf plave boje</p>

Atribut lang

Jezik dokumenta moze biti deklarisan u html tagu. Jezik je deklarisan atributom lang Deklaracija jezika je vazna za citace ekrana kao i za pretrazivace.

<html lang="en-US"></html>

Atribut title

U ovom primeru smo atribut title dodali paragrafu. Tekst koji se nalazi u atributu ce biti prikazan onda kada kursor misa dovedemo preko teksta koji se nalazi u paragrafu.

<p title="Ovo je neko obavestenje">Ovo je paragraf</p>

Atribut id

Da bi definisali poseban stil za odredjeni element dodacemo mu atribut id.

<p id="plavi">Ovo je plavi tekst</p>

Zatim definisemo stil sa odredjenim atributom id.

#plavi { color: blue; }

Atribut class

Da bi definisali stil za posebnu vrstu elementa dodajemo atribut class.

<p class="plavi">Ovo je plavi tekst</p> .plavi { color: blue; } (CSS)

HTML <head> element

HTML <head> element nema nikakve veze sa naslovom vec je to kontejner za meta podatke.
Meta podaci su podaci koji opisuju HTML dokument i nisu vidljivi u pretrazivacu (korisnicima) a nalaze se izmedju tagova <html> i <body>.

<head> <title>Ovde se nalaze meta podaci</title> <meta charset="utf-8"> </head>

HTML Formatiranje teksta

HTML naslovi i paragrafi se mogu menjati pomocu specijalnih tagova:

<b> - Boldovani tekst
<strong> - Veoma boldovan tekst
<i> - Zakrivljen tekst
<em> - Naglasen tekst
<mark> - Markiran tekst
<small> - Mala slova
<del> - Prekrizen tekst
<ins> - Podvucen tekst
<sub> - Subscript tekst
<sup> - Superscript tekst

HTML Styles

Podesavanje stila na HTML elementu moze se postici pomocu atributa style.
Atribut style ima sledecu sintaksu:

<ime_taga style="svojstvo: vrednost;">

Primer - Svojstvo background-color definise boju pozadine kod html elementa.

<body style="background-color: gray;">

HTML Text Color

Svojstvo color odredjuje boju html elementa

<h1 style="color: red;">Ovo je naslov crvene boje</h1> <p style="color: powderblue;">Ovo je paragraf svetlo plave boje</p>

HTML Fontovi

Svojstvo font-family definise font slova koja koristimo u html-u.

<h2 style="font-family: verdana;">Ovo je neki naslov h2</h2> <h3 style="font-family: courier;">Ovo je neki drugi naslov</h3>

HTML Velicina teksta

Svojstvo font-size definise velicinu teksta u html elementu.

<h1 style="font-size: 500%;">Ovo je naslov</h1> <p style="font-size: 300%;">Ovo je paragraf</p>

HTML Text aligment

Svojstvo text-align definise horizontalno poravnanje teksta HTML elementa.

<h1 style="text-align: center;">Ovaj naslov je centriran</h1> <p style="text-align: center;">Ovaj paragraf je centriran</p>

HTML element <abbr> za skracenice

Element <abbr> definise skracenicu ili akronim.
Skracenice mogu da daju veoma korisne informacije pretrazivacima.

<p>Nasa <abbr title="Eleketrotehnicka skola-Mihajlo Pupin">ETS</abbr>Osnovana je 1950. godine</p>

HTML element <adress>

HTML element <adress> definise podatke o autoru ili vlasniku teksta ili dokumenta, i obicno je prikazan kurzivom (italic).
Vecina pretrazivaca ce dodati prelomne linije pre i posle elementa.

<adress> Pisac Milan Simic. Posetite web sajt: www.ets.edu.rs Zemunska BB, Gornja Gusterica Srbija </adress>

HTML komentari

Da bi u svom html dokumentu dodali komentar koristicemo sledecu sintaksu:

<!--Ovo je komentar-->

Obratite paznju da je na pocetku taga znak ! a na kraju ga nema.
Pretrazivac ne prikazuje komentare ali su oni bitni da bi se dokumentovao html kod i pomazu pri pisanju podsetnika i ostalih komentara za odrzavanje koda.

<p>Ovo je paragraf</p> <!--Podestnik o ostalim informacijama-->

Stilizovanje HTML-a pomocu CSS-a

- CSS je standard za Cascading Style Sheet.
- CSS opisuje kako ce html biti prikazan u pretrazivacu, na papiru ili nekom drugom medijumu.
On moze biti dodat u html na 3 nacina:

  1. Inline - pomocu atributa style
  2. Internal - koristeci <style> element u <head> sekciji.
  3. External - koristeci eksterni CSS fajl.

Inline CSS se koristi za primenu jednostavnog stila na jedan html element.

<h1 style="color: blue;">Ovo je naslov plavi</h1>

Ovaj primer pokazuje kako se preko CSS-a menja boja nekog (h1) elementa.

Internal CSS se koristi za definisanje stila za jednu HTML stranicu i definisan je u odeljku <head> unutar elementa <style>

<style> body { background-color: powderblue; } h1 { color: blue; } p { color: red; } </style>

External CSS je nacin koji se koristi najcesce i sluzi za definisanje vise stranica.

Boje linkova

Sto se tice boje linkova ovako bi se one pokazale u pregledacu podrazumevano:

  • link koji nije posecen je plave boje i podvucen je
  • link koji je posecen je podvucen i ljubicaste je boje
  • aktivan link je crvene boje i podvucen je

Sve te podrazumevane vrednosti mozemo promeniti pomocu CSS-a

<style> a:link { color: green; *Za sam link background-color: transparent; text-decoration: none; } a:visited { color: pink; *Kada je link posecen background-color: transparent; text-decoration: none; } a:hover { color: red; *Kada misem predjemo preko linka background-color: transparent; text-decoration: underline; } a:active { color: yellow; *Kada je link u toku otvaranja background-color: transparent; text-decoration: underline; } </style>

CSS svojstva okvira

CSS border svojstvo definise okvir oko html elementa

p { border: 1px solid blue; }

CSS padding svojstvo definise prostor izmedju teksta i okvira

p { border: 1px solid blue; padding: 30px; }

CSS margin svojstvo definise prostor izvan okvira

p { border: 1px solid blue; margin: 50px; }
